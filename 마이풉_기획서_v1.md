## 탄생 배경

의사들이 자기 사는 곳 잘 말 안하고, 연락처 꼭꼭 숨긴다고 함.
왜냐하면 같은 동네 사는 사람들이 애기 아픈 거 같다고 급하게 잠깐만 와달라고 그런다고…. 방문 진찰 서비스 돈내고 받을 생각은 아닌 거 같은데, 자기 아이 똥 사진 봐달라고 갑자기 똥사진 보내는 부모들도 많다고.

갑자기 남의 똥 사진 받는 의사들의 정신과 안구 건강이 안좋을 것 같아 딱하고, 사실 자기 애 걱정하는 엄마들 마음도 이해는 가서…
또 쓰레드의 누군가는 자기 똥을 AI한테 보여주고 건강상태 평가해 달라고 그러던데…
나는 내 똥조차 보는 것도 싫은데… 여러 인사이트를 종합하여

**그래서 만들게 됨! 마이풉! 웹앱**

## 1. 서비스 개요

사용자가 대변을 직접 보지 않아도 AI가 건강을 분석해주는 모바일 웹앱.
카메라 화면에서 변기/기저귀 인식 시 자동으로 모자이크 또는 캐릭터 스티커로 가림 처리.
성인 모드(My 💩)와 베이비 모드(My Baby 💩) 두 가지 제공.

### 핵심 가치

- **안구보호**: 카메라 켜지는 시점부터 모자이크 적용. AI만 원본을 봄
- **간편 건강 체크**: 촬영 즉시 별점 + AI 코멘트
- **이상 감지 시 추가 질문**: 형태/색 안 좋으면 AI가 뭐 먹었는지, 약 복용 여부 등 추가 확인
- **모두를 위한 서비스**: 환자, 어르신, 초보 부모 등 누구나 사용 가능. 로그인 없이 게스트로도 사용 가능

## 2. 디자인 시스템

### 색상

| 용도 | 색상명 | HEX | RGB |
| :--- | :--- | :--- | :--- |
| 배경 | 크리미 바닐라 | `#FAF7F2` | rgb(250, 247, 242) |
| 키컬러 | 소프트 라벤더 | `#B8A9C9` | rgb(184, 169, 201) |
| 서브 액센트 | 웜 베이지 | `#D4B896` | rgb(212, 184, 150) |
| 텍스트 | 소프트 다크 브라운 | `#3D3535` | rgb(61, 53, 53) |

**금지 색상**: 분홍, 파랑, 그린 계열 (음식 연상 방지)
**컨셉**: 소프트아이스크림 같은 부드러운 색감

### 캐릭터

- **성인 응가**: 힙한 선글라스 착용, 쿨한 느낌
- **베이비 응가**: 통통하고 귀여운 느낌, 튼튼! 귀욥! 할 수 있다!
- 각 캐릭터별 스티커 4~6종 (똥 가리는 용도)
- 추후 캐릭터 IP 사업 여지 확보

### 언어

기기 설정 언어 자동 동기화. 수동 설정은 **한국어 / 영어** 2개

## 4. DB 스키마

### users

```
users/{userId}
  ├── uid              string
  ├── email            string
  ├── loginType        enum  [guest, email, google, kakao]
  ├── createdAt        timestamp
  ├── mode             enum  [adult, baby, both]
  │
  ├── cameraSettings
  │     ├── coverType      enum  [mosaic, sticker]
  │     ├── stickerSet     enum  [adult, baby]
  │     └── coverEnabled   boolean
  │
  └── profile
        ├── adult
        │     ├── name          string
        │     ├── birthYear     number
        │     ├── gender        enum  [M, F, N]
        │     ├── conditions    array   // 유당불내증, 암환자 등
        │     └── medications   array
        └── baby
              ├── name          string
              ├── birthDate     timestamp
              ├── feedType      enum  [breast, formula, mixed, weaning]
              └── weaningStart  timestamp
```

### records

```
records/{recordId}
  ├── userId           string
  ├── mode             enum  [adult, baby]
  ├── createdAt        timestamp
  ├── imageRef         string      // Firebase Storage 원본 경로
  ├── imageCoveredRef  string      // 모자이크/스티커 처리된 버전
  ├── coverEnabled     boolean
  │
  ├── analysis
  │     ├── score          number   // 0.0 ~ 5.0 (0.5 단위)
  │     ├── comment        string   // AI 코멘트
  │     ├── drip           string   // 건강할 때 드립 멘트 (랜덤 조합)
  │     ├── color          string   // 색상 분류
  │     ├── shape          string   // 형태 분류 (브리스톨 척도)
  │     ├── alertLevel     enum     [normal, caution, warning]
  │     └── followUpQs     array    // AI 추가 질문 목록
  │
  └── userInput
        ├── meal           string
        ├── medication     string
        └── memo           string
```

### analytics (비식별 빅데이터용)

```
analytics/{docId}
  ├── createdAt        timestamp
  ├── mode             enum  [adult, baby]
  ├── score            number
  ├── color            string
  ├── shape            string
  ├── feedType         string    // 베이비만
  ├── ageGroup         string    // 10대/20대/30대...
  └── region           string    // 국가 코드만
  // userId, 이미지 없음. 개인 식별 불가
```

## 5. 화면 구성

탭 3개: **카메라 / 히스토리 / 설정**

---

### 탭 1 — 카메라 (기본 진입)

#### 온보딩 (최초 1회)

- `My 💩` / `My Baby 💩` 두 버튼 선택
- 선택값 LocalStorage 저장, 이후 스킵
- 좌상단 `< 이전` 버튼 항상 표시

#### 촬영 플로우

```
카메라 열림
  → 변기 / 기저귀 인식
  → 해당 영역 실시간 가리기 (설정값 기준: 모자이크 or 스티커)
  → 촬영 버튼
  → 로딩 (AI 분석 중)
  → 결과 화면
        ├── 별점 (★ 0~5, 0.5단위)
        ├── AI 코멘트
        ├── 건강이면 드립 멘트 랜덤 출력
        └── 이상이면 추가 질문 팝업
              "오늘 뭐 드셨어요?"
              "약 드시는 거 있으세요?"
              "유당불내증 있으세요?"
```

#### 카메라 화면 UI 요소

- **우상단**: 가리기 온오프 토글 (빠른 전환용)
- **우상단 토글 옆**: 모자이크 ↔ 스티커 전환 버튼
- **가리기 기본값**: ON (설정에서 기본값 변경 가능)

모자이크/스티커 선택 기본값은 설정 탭에서 지정.
카메라 화면에서는 빠르게 전환만 가능.

---

### 탭 2 — 히스토리

#### 상단: 꺾은선 그래프

- X축: 날짜 (7일 or 30일)
- Y축: 별점 (0~5)
- 점 누르면 해당 날짜 기록 상세 팝업
- 별점 트렌드로 장 컨디션 변화 한눈에 파악

#### 중단: 캘린더 뷰

- 날짜별 색상 도트 표시
  - 🟢 녹색: 양호 (별점 4 이상)
  - 🟡 노랑: 주의 (별점 2.5~3.5)
  - 🔴 빨강: 경고 (별점 2 이하)
- 날짜 누르면 해당 기록 상세 보기

#### 하단: AI 7일 총평

매번 새로 호출. 예시:
- "장 컨디션이 좋으시네요! 👍"
- "장 건강 빨간 신호! 전문의와 상담하세요 🚨"

#### 뷰 전환

- 7일 보기 / 30일 보기 탭 전환
- 기준: 오늘 날짜, 맨 오른쪽이 오늘

#### 모드 전환

- 내 똥 ↔ 베이비 똥 가로 스와이프로 전환

---

### 탭 3 — 설정

```
├── 가리기 설정
│     ├── 기본값: 모자이크 or 스티커 선택
│     ├── 스티커 선택 시: 성인 응가 / 베이비 응가 캐릭터
│     └── 가리기 기본 ON/OFF 설정
│
├── 마이 건강 정보 등록/수정
├── 마이 베이비 건강 정보 등록/수정
│     └── 베이비풉 촬영 후 바로 여기로 유도
│
├── 다운로드
│     ├── 전체 다운로드
│     ├── 날짜 선택 다운로드
│     └── 기간 설정 다운로드
│           └── 5,000원 결제 후 이용
│                 (별점 + AI코멘트 + 식사메모 포함된 PDF)
│
├── 언어 설정 (한국어 / English)
│
├── 로그인 / 로그아웃
│     └── 게스트 상태면 회원가입 유도
│           (이메일 / Google / 카카오)
│
└── 개인정보 처리방침 / AI 학습 데이터 활용 동의
```

## 7. 개인정보 및 데이터 정책

- 원본 이미지: Firebase Storage 암호화 저장, 본인만 접근 가능
- AI 학습용: 개인 식별 정보 제거 후 별도 analytics 컬렉션 저장
- 동의 구조: 서비스 이용 동의 / AI 학습 활용 동의 (선택) 분리
- 탈퇴 시 원본 이미지 및 개인정보 즉시 삭제
- analytics 데이터는 비식별 처리 후 보존
